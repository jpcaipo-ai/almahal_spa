# Almahal Spa Growth Dashboard

Dashboard local/publicable para seguimiento de ventas, asesoras, lifecycle de clientes y calidad de conversaciones.

## Archivos principales

- `index.html`: entrada del tablero.
- `styles.css`: estilos del tablero.
- `app.js`: logica de filtros, metricas y visualizaciones.
- `sales-data.js`: data de ventas normalizada.
- `quality-data.js`: data de auditorias conversacionales.

## Corte de data

- Ventas actualizadas al 30/06/2026.
- Conversaciones actualizadas al 28/06/2026.

Si se usa con GitHub Pages, publicar desde la rama `main` y carpeta root.
